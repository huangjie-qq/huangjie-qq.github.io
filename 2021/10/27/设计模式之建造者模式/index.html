设计模式之建造者模式

介绍：

建造者模式的定义是：将复杂对象的构造与它的表示分离，使同样的构建过程可以创建不同的表示，这样的设计模式被称为建造者模式。

为什么使用构建者模式？什么场景使用构建者模式？

  • 假如我们需要一个对象，一个对象的创建非常的复杂，对象中还要对象的封装，那么我们就可以使用构建者模式，将该对象的构建交给它来处理，我们使用的时候找他要一个对象就可以了。

假如现在有个User类如下

/**
 * @Author：小球某
 * @Date：2021/10/27/17:48
 */
@Data
public class User {

    private String name;
    private int age;
    private String sex;
    private String phone;
    private String address;

}

当我们要构建name字段和age字段时 我们可以这样

public User(String name, int age) {
        this.name = name;
        this.age = age;
}

同理当需要构建 name字段 age字段 sex字段时

public User(String name, int age, String sex) {
        this.name = name;
        this.age = age;
        this.sex = sex;
}

当我们每次需要构配不同对象时，就要为这些字段生成不同的构造函数，如此一来 我们的构造函数就特别多

并且阅读性也特别的不好。

当然还有一种解决办法 就是生成 get set方法

   public void setName(String name) {
        this.name = name;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public void setSex(String sex) {
        this.sex = sex;
    }

    public void setPhone(String phone) {
        this.phone = phone;
    }

    public void setAddress(String address) {
        this.address = address;
    }

写的时候这样就可以了

public static void main(String[] args) {
      User user=new User();
      user.setName("小球某");
      user.setAge(20);
      user.setSex("男");
}

这样一来就痛苦的是我们程序员了 这个时候构建者就来了

首先来个静态内部类 字段和User类保持一致

/**
     * 内部类
     */
    public static class builder{
    
        private String name;
        private int age;
        private String sex;
        private String phone;
        private String address;

    }

提供构建各个字段的方法

     public builder name(String name){
         this.name=name;
         return this;
     }

     public builder age(int age){
         this.age=age;
         return this;
     }

     public builder sex(String sex){
         this.sex=sex;
         return this;
     }

     public builder phone(String phone){
         this.phone=phone;
         return this;
     }

     public builder address(String address){
         this.address=address;
         return this;
     }

最后提供构建build方法 同时在User提供构造函数

public User build(){
   return new User(this);
}

public User(builder builder){
        this.name= builder.name;
        this.age= builder.age;
        this.sex= builder.sex;
        this.phone= builder.phone;
        this.address= builder.address;
 }

完整代码

/**
 * @Author：小球某
 * @Date：2021/10/27/17:48
 */
@Data
public class User {

    private String name;
    private int age;
    private String sex;
    private String phone;
    private String address;


    public User() {
    }

    public User(builder builder){
        this.name= builder.name;
        this.age= builder.age;
        this.sex= builder.sex;
        this.phone= builder.phone;
        this.address= builder.address;
    }

    /**
     * 内部类
     */
    public static class builder{
    
        private String name;
        private int age;
        private String sex;
        private String phone;
        private String address;

        public builder name(String name){
            this.name=name;
            return this;
        }

        public builder age(int age){
            this.age=age;
            return this;
        }

        public builder sex(String sex){
            this.sex=sex;
            return this;
        }

        public builder phone(String phone){
            this.phone=phone;
            return this;
        }

        public builder address(String address){
            this.address=address;
            return this;
        }

        public User build(){
            return new User(this);
        }

    }
}

用法

public static void main(String[] args) {
        User user1 = new User.builder().name("小球某").age(20).build();
        User user2 = new User.builder().name("小球某").age(20).sex("男").build();
        System.out.println(user1);
        System.out.println(user2);
}

运行结果如下：

User(name=小球某, age=20, sex=null, phone=null, address=null)
User(name=小球某, age=20, sex=男, phone=null, address=null)

终极方法：lombok的Builder注解

@Data
@Builder
@AllArgsConstructor
@NoArgsConstructor
public class User {

    private String name;
    private int age;
    private String sex;
    private String phone;
    private String address;
    
}

用法

 public static void main(String[] args) {
        User user = User.builder().name("小球某").age(20).sex("男").build();
        System.out.println(user);
 }

运行结果如下：

User(name=小球某, age=20, sex=男, phone=null, address=null)
